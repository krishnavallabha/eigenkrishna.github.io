SerenityCare

A full-stack hospital management system built with Flask and Vue.js, featuring real-time appointment booking, multi-role access (Admin, Doctor, Patient), treatment history tracking, and background task automation using Redis and Celery.

The system addresses critical challenges in healthcare administration by providing a centralized platform that eliminates scheduling conflicts, streamlines patient-doctor interactions, and reduces administrative overhead through intelligent automation.

Problem Statement
Manual appointment scheduling leads to double bookings and scheduling conflicts. Patients struggle to find doctors, book appointments, and access medical history. Administrative overhead increases due to fragmented systems and lack of centralized management.
Approach
Multi-role web application (Admin, Doctor, Patient) using Flask backend and Vue.js frontend. Real-time appointment booking with conflict prevention and treatment history tracking. Performance optimization through Redis caching and database indexing, with JWT-based security and role-based access control.
Tech Stack
Flask Vue.js Bootstrap SQLite Redis Celery JWT Flask-SQLAlchemy Flask-JWT-Extended Chart.js Axios ReportLab Twilio
Implemented Features
  • Authentication & Authorization - Multi-role login, JWT tokens
  • Admin Features - Dashboard, CRUD operations, analytics
  • Doctor Features - Appointment management, treatment records, availability
  • Patient Features - Registration, booking, history access
  • Appointment Management - Conflict prevention, status tracking
  • History & Reporting - Complete history with filters, exports
  • Performance - Redis caching, database indexing
  • PWA Support - Offline functionality, installable
  • Security - Password hashing, role-based access
  • Data Export - PDF/CSV generation
