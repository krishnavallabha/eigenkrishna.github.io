Slotly

A smart vehicle parking management system that allows users to view available slots, book them in real-time, and manage parking efficiently. The system simulates real-life parking scenarios to improve urban mobility and reduce congestion.

It is a multi-user application that manages different parking lots, parking spots, and parked vehicles, specifically designed for 4-wheeler parking. The system follows MVC architecture and provides comprehensive analytics for both users and administrators.

Objective
Design and develop a smart vehicle parking system that allows users to view available slots, book them in real-time, and manage parking efficiently. The system should simulate or reflect real-life parking scenarios to improve urban mobility and reduce congestion.
Architecture
The system follows the MVC (Model-View-Controller) architecture. Controllers manage the logic and routing using Flask. Templates (HTML + CSS) are used for front-end layout and rendering views. Models handle database interactions for users, reservations, and parking lot data.
Tech Stack
Flask Jinja2 HTML CSS Bootstrap SQLite
Features

User Features

  • Dashboard showing available parking lots
  • Search parking lots by pin code, address, or name
  • View rate per hour for each lot
  • Personal analytics including total reservations, amount spent, monthly expenditure
  • Average parking duration tracking
  • Monthly spend trend visualization (graph)
  • Usage by day of the week analytics
  • Favorite parking lot identification
  • Peak usage hours analysis
  • View complete parking history
  • Edit profile option
  • Book / Release a parking spot
  • Extend parking lot reserved time

Admin Features

  • Dashboard with Home, Dashboard, Users, Sales, Summary sections
  • Parking Lot & Spot Management - View and search parking lots
  • Add / Update lots with full control
  • Manage spot status (Available / Reserved)
  • Edit maximum spots, rate per hour, area, lot name, and address
  • User Section - View registered users with username, name, email
  • Track total reservations and active reservations per user
  • Sales Section - Total sales (All-time) and sales this month
  • Monthly sales trend visualization
  • Monthly reservation count tracking
  • Recent reservations overview
  • Summary Section - Total number of parking lots and overview analytics
